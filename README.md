# Stock Monitor

This repository is a sanitized public copy of a private stock monitoring
project.

The public copy intentionally redacts:

- Numeric strategy windows, thresholds, formulas, and ranking parameters
- Ticker lists, market universes, and instrument mappings
- Runtime state, holdings, execution logs, and cached market data
- Notification keys, account details, and local environment paths

Runtime configuration and private operating parameters are not distributed here.
